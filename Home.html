Welcome to Web Application Development Frameworks

We’re excited to have you in this class!

This course explores various architectural and design factors that have to be taken into consideration when building complex web applications. Students will get to appreciate the importance of having web applications designed in accordance with industry best practices. To implement these best practices, suitable web development frameworks will be put to use. A web application will typically use more than one framework, forming a stack of frameworks that cover both the front-end and the back-end. This course only seeks to cover the most prevalent framework stacks at a given point in time.
